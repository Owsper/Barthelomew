# 🧸 Bartholomew Huggy

**A wholesome browser extension that brings a cuddly teddy bear straight to your screen.**

Click the toolbar icon and watch Bartholomew pop up on any webpage to give you a quick hug when you need it most.



### 🧸 Meet Bartholomew

Bartholomew Huggy is your friendly desktop companion. Whether you're stressed, bored, or just need a little smile — he's always ready for a hug.

### Installation

1. Download or clone this repository
2. Open your browser and go to `chrome://extensions/` (or `edge://extensions/`)
3. Enable **Developer mode** (top right)
4. Click **"Load unpacked"** and select the extension folder
5. Pin Bartholomew Huggy to your toolbar
6. Start spreading hugs! 🧸💕

### How to Use

- Click the teddy bear icon in your browser toolbar
- Bartholomew will appear with a cute animation
- Click on him to give him a hug and see floating hearts
- Click anywhere else on the page to dismiss him

### Tech Stack

- Manifest V3
- Chrome Extensions API (`chrome.action`, `chrome.scripting`)
- Vanilla JavaScript + CSS Animations
- No external dependencies
